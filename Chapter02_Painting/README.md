# 페인팅 붓 크기 조절 기능 추가

이 프로젝트는 OpenCV의 마우스 콜백 함수와 키보드 이벤트를 활용하여, 불러온 이미지 위에 직접 마우스로 그림을 그릴 수 있는 간단한 캔버스 예제입니다.

## 📌 주요 기능
* **마우스 드로잉**: 마우스 클릭 및 드래그를 통해 이미지 위에 연속적인 선(원형 브러시)을 그릴 수 있습니다.
* **색상 변경**: 마우스 왼쪽 버튼과 오른쪽 버튼에 각각 다른 색상이 할당되어 있습니다.
* **브러시 크기 조절**: 키보드 입력을 통해 실시간으로 브러시(붓)의 굵기를 조절할 수 있습니다.
* **프로그램 종료**: 특정 키를 눌러 프로그램을 안전하게 종료합니다.

## 🛠️ 요구 사항 
코드를 실행하기 전에 아래 라이브러리가 설치되어 있어야 합니다.

```bash
pip install opencv-python numpy
```

## 🕹️ 조작 방법 (Controls)

| 조작 | 기능 설명 |
|---|---|
| **마우스 좌클릭 + 드래그** | 🔵 파란색 브러시로 그리기 |
| **마우스 우클릭 + 드래그** | 🔴 빨간색 브러시로 그리기 |
| **키보드 `+`** | 붓 크기 증가 (최대 15) |
| **키보드 `-`** | 붓 크기 감소 (최소 1) |
| **키보드 `q`** | 프로그램 종료 |

## 📝 주요 코드 설명

```python
def mouse_event(event, x, y, flags, param):
# ... (중략) ...
cv.setMouseCallback("Paint", mouse_event)
```
> 🖱️ 마우스 클릭, 드래그, 이동 등의 이벤트가 발생할 때마다 자동으로 실행될 `mouse_event` 함수를 정의합니다. 그리고 `cv.setMouseCallback()`을 이용해 "Paint"라는 이름의 창과 이 함수를 연결해 줍니다.

```python
cv.circle(img, (x, y), brush_size, color, -1)
```
> 🎨 드래그 상태(`drawing == True`)일 때, 마우스 커서의 현재 위치 `(x, y)`에 `cv.circle`을 이용해 색칠된 원(`-1`)을 연속적으로 그려서 선처럼 보이게 만듭니다.

```python
while True:
    cv.imshow("Paint", img)
    key = cv.waitKey(1)
    
    if key == ord('+'):
        brush_size = min(15, brush_size + 1)
```
> 🔄 `while True` 무한 루프를 돌면서 변경된 `img`를 실시간으로 화면에 업데이트합니다. `cv.waitKey(1)`을 통해 1ms마다 키보드 입력을 감지하고, `+`나 `-` 키가 눌리면 브러시 사이즈를 즉각적으로 조절합니다. (`min`, `max` 함수로 최소/최대치 제한)
